# 🎉 年会抽奖活动小程序

一个功能丰富、界面美观、互动性强的年会抽奖小程序。

## ✨ 功能特点

- 🎯 **转盘抽奖**：精美的转盘动画，支持自定义转盘速度
- 👥 **参与者管理**：轻松添加、删除参与者，支持批量导入
- 🏆 **获奖记录**：自动记录获奖名单，支持清空记录
- 🎨 **精美UI**：现代化渐变设计，流畅的动画效果
- 📱 **响应式设计**：完美适配各种屏幕尺寸
- 🎊 **彩纸特效**：获奖时自动播放彩纸动画
- 🔊 **音效支持**：可开启/关闭抽奖音效
- ⚙️ **灵活设置**：可设置每次抽取人数、转盘速度等

## 🚀 快速开始

1. **打开程序**
   - 直接在浏览器中打开 `index.html` 文件即可使用
   - 无需安装任何依赖或服务器

2. **添加参与者**
   - 在输入框中输入姓名，按回车或点击"添加"按钮
   - 或点击"批量导入"按钮，每行输入一个姓名

3. **开始抽奖**
   - 点击"开始抽奖"按钮
   - 转盘会自动旋转并随机选择获奖者
   - 获奖者会显示在弹窗中，并自动添加到获奖名单

4. **查看获奖名单**
   - 所有获奖者会显示在右侧的"获奖名单"区域
   - 可以随时清空获奖记录

## 📖 使用说明

### 添加参与者
- **单个添加**：在输入框中输入姓名，按回车键或点击"添加"按钮
- **批量导入**：点击"批量导入"按钮，在弹出的对话框中每行输入一个姓名，点击"确认导入"

### 抽奖设置
- **每次抽取人数**：设置一次抽奖抽取多少人（1-10人）
- **转盘速度**：调整转盘旋转的速度（1-10档）
- **音效开关**：开启或关闭抽奖时的音效提示

### 获奖管理
- 获奖者会自动添加到获奖名单中
- 获奖名单按顺序显示，支持滚动查看
- 可以点击"清空"按钮清空所有获奖记录

## 🎨 界面特色

- **渐变背景**：紫色渐变背景，营造节日氛围
- **流畅动画**：所有操作都有平滑的动画过渡效果
- **转盘特效**：转盘旋转带有缓动效果，更加真实
- **彩纸庆祝**：获奖时自动播放彩纸动画效果
- **响应式布局**：自适应不同屏幕尺寸，移动端友好

## 💡 使用技巧

1. **批量导入**：如果参与者较多，建议使用批量导入功能，可以快速添加所有参与者
2. **多次抽奖**：可以连续进行多次抽奖，每次抽奖都会记录在获奖名单中
3. **重置转盘**：点击"重置"按钮可以将转盘恢复到初始位置
4. **移除参与者**：点击参与者卡片上的"×"按钮可以移除该参与者

## 🔧 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画效果
- **JavaScript (ES6+)**：交互逻辑和抽奖算法
- **Canvas API**：转盘绘制
- **Web Audio API**：音效生成

## 📝 注意事项

- 建议使用现代浏览器（Chrome、Firefox、Edge、Safari等）
- 如果参与者数量为0，转盘会显示提示信息
- 抽奖过程中无法添加或删除参与者
- 获奖者会从参与者列表中移除（可选功能）
